# 📊 Приложение для управления финансами

Консольное приложение на Java для отслеживания доходов, расходов и бюджетирования.

## 🚀 Возможности
- 📝 Регистрация и авторизация пользователей
- 💰 Добавление доходов и расходов по категориям
- 📊 Установка и отслеживание бюджета
- 📈 Статистика и аналитика
- 💾 Автосохранение данных в JSON

## 🛠 Технологии
- Java 8+
- Jackson для работы с JSON
- JUnit 5 для тестирования

## 📦 Установка и запуск

### Требования
- Java JDK 8 или новее
- Maven

### Установка
```bash
git clone <ваш-репозиторий>
cd finance-management-app
mvn clean compile
